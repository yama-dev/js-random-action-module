## Using

### NPM Usage

``` bash
# install npm.
npm install --save-dev js-random-module
```

``` javascript
// import.
import RANDOM_MODULE from 'js-random-module';
```

### Basic Standalone Usage

``` html
<script src="./js-random-module.js"></script>
<script>
</script>
```
